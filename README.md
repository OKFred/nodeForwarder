# nodeForwarder
## 将你的请求转发到另一个主机上。
前端调试必备。😎

假设你在电脑上开了一台虚拟机，但是局域网的同事访问不到？
当然你可以安装Nginx或者FRP配置下。
或者你可以试试这个。
输入你需要反代的地址，然后回车即可启动。一键转发你的请求。
